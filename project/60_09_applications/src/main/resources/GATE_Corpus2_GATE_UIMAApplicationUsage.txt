
GATE_Corpus2_GATE_UIMAApplication

Description
The program GATE_Corpus2_GATE_UIMAApplication reads in a GATE corpus, transforms each of the files
to gate xml and uima xml files. 

By default, the de-identification option is on to strip the redacted formatted forms to spaces.

O

Program
java -jar $FRAMEWORK_HOME/10_libs/GATE_Corpus2_GATE_UIMAApplication-jar-with-dependencies.jar 
    --gateHome=
    --inputDir=
    --outputDir=
    --outputTypes=
